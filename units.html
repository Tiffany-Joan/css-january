CSS units & Measurements

CSS provides various units to define dimensions, spacing and typography. These units are categorized into absolute and relative types.

1. Absolute Units

Absolute units are defined fixed sizes that do not change relative to the viewport or parent elements.

  • px - pixels
  • cm - centimeters
  • mm - millimetres
  • in - inches

2. Relative Units

Relative units depend on the parent elements, viewport sizes, or root elements, making them ideal for responsive design

em and rem are font related units. em unit is relative to the parent element font size and rem is relative to root element(html)font size

Percentage(%) units are relative to the parent element(width or height)

vw (viewport width) and (viewport height) units are relative to the viewport/available/
